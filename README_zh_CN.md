# Rig Bridge / 动作搬运工

[English](README.md)

Rig Bridge（动作搬运工）是一个 Blender 人形动作自动传递插件。它可以把一套人形骨架上的动画自动搬到另一套骨架上；插件优先识别已知骨架规范，没有命中预设时，会根据语义名称、骨骼层级、身体几何、静止姿态、左右结构和身体前向继续判断。

## 主要功能

- 自动识别动作骨架和目标骨架
- 预设优先，无名称结构识别回退
- 通过左右肢体、躯干终点、脚趾和完整手指链识别匿名骨架
- 单动作和集合批量重映射
- 根位移动作与原地动作
- 静止姿态和身体前向门禁
- 批量失败自动清理，生成 Action 可保存重开

## 使用要求

- Blender 4.2 至 5.2
- 两套有效人形骨架

## 安装

可以从 [Blender Extensions 官方页面](https://extensions.blender.org/add-ons/humanoid-remap-studio/) 安装 Rig Bridge，也可以在 Blender 的“获取扩展”中搜索 `Rig Bridge`。

## 快速开始

1. 在 3D 视图侧边栏打开独立的“重映射”标签。
2. 选择单个动作骨架，或选择包含多个动作骨架的集合。
3. 选择目标人形骨架。
4. 点击“自动识别”。
5. 根据需要选择“原地动作”。
6. 点击“执行重映射”或“执行批量”。

需要重新测试时，可以先使用“清除结果”删除插件生成的 Action。

## 隐私与权限

插件不申请网络、文件、剪贴板、摄像头或麦克风权限。

## 包结构

运行时代码按动作处理、骨架识别、人形画布、重映射、操作器、界面和翻译拆分为独立模块。包入口只负责注册和场景属性，内部统一使用相对导入；开发期重载遵循 Blender 官方扩展生命周期，不包含自定义模块重载器。

## 兼容说明

Rig Bridge 使用自身代码完成识别和重映射，不会调用其他插件的 operator 或运行时属性。由常见绑定工具生成的骨骼结构仍可作为普通输入数据被识别。
