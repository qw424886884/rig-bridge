# Security Policy

## Supported Version

Security fixes are provided for the latest published release.

## Reporting a Vulnerability

Do not publish sensitive scene data, private character files, credentials, or exploit details in a public issue. Contact the maintainer through the GitHub profile at <https://github.com/qw424886884> and provide a minimal reproduction that does not contain private assets.

The extension does not require network access. Any future feature that adds network access must declare the permission in `blender_manifest.toml` and document the destination and purpose before release.
